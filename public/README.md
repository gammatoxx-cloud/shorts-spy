# Public Assets Directory

Place your logo image file here as `logo.png`.

The logo will be used in:
- Main navigation header (ConditionalHeader component)
- Dashboard sidebar (DashboardSidebar component)

**File requirements:**
- Name: `logo.png`
- Location: `/public/logo.png`
- Format: PNG, JPG, or SVG (PNG recommended)
- Recommended size: Square format (e.g., 512x512px or 1024x1024px) for best quality

Once you add the file, it will automatically appear in the app.
